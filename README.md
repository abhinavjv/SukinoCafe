## ✨ About the Project

This is the official website for **Sukino Cafe & Kitchen**, a beloved 
coffee shop located in Banashankari, Bengaluru. The site is designed 
with a warm, editorial aesthetic — dark espresso tones, gold accents, 
and smooth animations — to reflect the cafe's intimate and artisanal vibe.

---

## 🚀 Features

- 🗓️ **Table Reservation System** — Book a table in real time
- 🛒 **Online Ordering** — Browse menu and place orders instantly  
- 📋 **Dynamic Menu Page** — Menu powered live from database
- 📍 **Location & Info** — Address, hours, and contact details
- ⭐ **Customer Reviews** — Showcasing our 4.5★ rating

---


## 🗄️ Database Structure (Supabase)
```
reservations   → stores all table bookings
orders         → stores all online orders
menu_items     → stores live menu data
```
